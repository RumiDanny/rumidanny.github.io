how to use:

Installation

  • Clone the repository: git clone https://github.com/RumiDanny/Rada
  • Navigate to the project directory: cd Rada
  • Install the required packages: pip install -r requirements.txt

Usage

  • To run the application, execute the following command: python3 main.py
